Section 1.3 Combinations

In the previous section, Permutations, order mattered. However, in Combinations, order does not matter.

Example 1.3.1.

How many different ways can we combine 3 elements of the set \(\{1,2,3,4\}\text{?}\)

Solution.

\(\{1,2,3\}, \{2,3,4\}, \{1,2,4\}, \{1,3,4\}\text{.}\) There are 4 different ways to arrange them.

Since order does not matter, \(\{1,2,3\} = \{3,2,1\}\text{.}\) Keep this in mind as we introduce some formulas.

In Example 1, we are choosing 3 elements from a set with 4 elements. If we are dealing with larger sets, listing out all possibilities would take too long. Instead, we can use the following defintion.

Definition 1.3.2.

We define \({n \choose r}\) for \(0 \leq r \leq n\) by,

\begin{equation*} {n \choose r} = \frac{n!}{(n-r)!r!} \end{equation*}

where \(n\) is the number of things we can choose from, and \(r\) is the number of things we want to choose. We read this as “n choose r.”

This is called the Binomial Coefficient. Why? We’ll find out later. First, let’s put this into practice on the next example.

Example 1.3.3.

Suppose you’re in a class full of 21 students and the teacher asks you to help her with a task. You need to find four other classmates to help. How many different groups can you form?

Solution.

\({n \choose r} = {20 \choose 4} = \frac{20!}{16!4!} = 4845\text{.}\)

We have 20 classmates (\(n\)), and we choose four (\(r\)) of them.

Let's use what we've learned in Sections 1.1 and 1.2 to solve the next example.

Example 1.3.4.

Imagine the colors in a candy bag are equally distributed. There is red, blue, green, and orange. If there was a way to differentiate between candies of the same color, how many different combinations of two candies of each other can we have if there are 64 total candies?

Solution.

\(64/4=16\) candies of each color.

\({16 \choose 2}{16 \choose 2}{16 \choose 2}{16 \choose 2}= (\frac{16!}{14!2!})^4 = 207360000\text{.}\)

Here's a useful fact:

\({n \choose r} = {n-1 \choose r-1} + {n-1 \choose r}\)

Let's think about why this is the case. Consider \(n\) objects in a box. Let 1 of the objects be red and the rest, \(n-1\) objects, be blue. Therefore, there are two possibilities, we pick the red object or we pick no red objects. If we pick the red object, this can be done in \({1 \choose 1}\) ways and we can pick the rest of the objects \({n-1 \choose r-1}\) ways. Hence, \({1 \choose 1}{n-1 \choose r-1}\text{.}\) If we pick no red objects, this can be done in \({n-1 \choose r}\) ways. Since these are mutually exclusive, we have the above identity.

Back to this Binomial Coefficient we mentioned earlier. It is an important piece of the Binomial Theorem.

Theorem 1.3.5. Binomial Theorem.

\begin{equation*} (x+y)^{n} = \sum_{k=0}^{n} {n \choose k} x^{k}y^{n-k} \end{equation*}

Think about this in expanding powers of sums.

Example 1.3.6.

Expand \((x+y)^2\)

Solution.

\((x+y)^2 = {2 \choose 0}x^0y^2 + {2 \choose 1}x^1y^1 + {2 \choose 2}x^2y^0 = y^2 + 2xy + x^2\)

Imagine we are choosing our \(x\) value. We can choose it 0 times, which leaves us with only one way to do this, hence the coefficient is 1. We can choose one \(x\text{,}\) which there are two ways to do this, hence the coefficient is 2. Similarly, we can choose two \(x\text{,}\) which there is only one way, and the coefficient is 1.

Section 1.3.1 Examples

Example 1.3.1.

Suppose we want to form a student-athlete committee at the university. We want this committee to representative of all sports, therefore, we pick two persons from each sport. There are 6 eligible baseball players, 12 eligible soccer players, 5 eligible football players, and 7 eligible lacrosse players. How many different committees can be formed?

Solution.

\({6 \choose 2}{12 \choose 2}{5 \choose 2}{7 \choose 2} = (15)(66)(10)(21) = 207900\)

Example 1.3.2.

Expand \((x+y)^n\text{,}\) where \(n = 3\text{.}\) Then, try when \(n =4\text{.}\)

Solution.

\((x+y)^3 = {3 \choose 0}x^0y^3 + {3 \choose 1}x^1y^2 + {3 \choose 2}x^2y^1 + {3 \choose 3}x^3y^0 = y^3 + 3xy^2 + 2x^2y^1 + x^3\)

\((x+y)^4 = {4 \choose 0}x^0y^4 + {4 \choose 1}x^1y^3 + {4 \choose 2}x^2y^2 + {4 \choose 3}x^3y^1 + {4 \choose 4}x^4y^0 = y^4 + 4xy^3 + 6x^2y^2 + 4x^3y + x^4\)
