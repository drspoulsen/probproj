In the previous section we talked about conditional probability. The probability of some event \(A\) given some event \(B\text{.}\) If events are independent, since one does not affect the other, we can define the conditional probability
\begin{equation*} P(A|B)=P(A) \end{equation*}
Furthermore,
\begin{equation*} P(B|A)=P(B) \end{equation*}
