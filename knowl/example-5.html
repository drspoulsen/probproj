Example 1.3.1.

How many different ways can we combine 3 elements of the set \(\{1,2,3,4\}\text{?}\)
Solution.
\(\{1,2,3\}, \{2,3,4\}, \{1,2,4\}, \{1,3,4\}\text{.}\) There are 4 different ways to arrange them.
Since order does not matter, \(\{1,2,3\} = \{3,2,1\}\text{.}\) Keep this in mind as we introduce some formulas.
