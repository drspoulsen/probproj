Example 1.3.2.

Expand \((x+y)^n\text{,}\) where \(n = 3\text{.}\) Then, try when \(n =4\text{.}\)
Solution.
\((x+y)^3 = {3 \choose 0}x^0y^3 + {3 \choose 1}x^1y^2 + {3 \choose 2}x^2y^1 + {3 \choose 3}x^3y^0 = y^3 + 3xy^2 + 2x^2y^1 + x^3\)
\((x+y)^4 = {4 \choose 0}x^0y^4 + {4 \choose 1}x^1y^3 + {4 \choose 2}x^2y^2 + {4 \choose 3}x^3y^1 + {4 \choose 4}x^4y^0 = y^4 + 4xy^3 + 6x^2y^2 + 4x^3y + x^4\)
